Что выведет программа? Объяснить вывод программы.

```go
package main

func main() {
	ch := make(chan int)
	go func() {
		for i := 0; i < 10; i++ {
			ch <- i
		}
	}()

	for n := range ch {
		println(n)
	}
}
```

Ответ:
```go
package main

// В отдельной горутине с помощью цикла значения отправляются в канал, из которого в основной
//горутине значения вычитываются и отправляются в stdout. После завершения цикла данные не будут
// больше отправляться, но цикл в основной горутине по-прежнему будет ждать значения и не дождется.
//Такая ситуация называется deadlock! Решение: отправить информацию о том, что значений
//больше не будет посредством закрытия канала!

func main() {
	ch := make(chan int)
	go func() {
		for i := 0; i < 10; i++ {
			ch <- i
		}
		//close(ch)              // Закрытие канала
	}()
	for n := range ch {
		println(n)
	}
}
```
